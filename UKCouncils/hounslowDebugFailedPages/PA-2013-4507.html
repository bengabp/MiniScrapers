Search planning applications

This service allows you to view the details of planning applications that have been submitted to the Planning Department. You can:

  • Monitor the progress of an application
  • View the drawings and associated documents submitted since January 2009, with details of determined applications gradually being added.
  • Submit comments about an application
  • Search a list of applications validated or decided each week
  • View whether an appeal has been lodged and any appeal decisions
  • View recent officer reports and all Decision Notices..
